`go-latest` is a pacakge to check version(tag) is latest.

If you're building tool in Golang, you can use this pacakge for encourage user to upgrade latest version of your tool. 

See more details in document at [https://godoc.org/github.com/tcnksm/go-latest](https://godoc.org/github.com/tcnksm/go-latest).

## Install

To install, use `go get`:

```bash
$ go get -d github.com/tcnksm/go-latest
```

1. Run `gofmt -s`
1. Create new Pull Request
